Welcome!

Have you ever wondered whether it's worth it to enable a specific graphics option in your favourite game? Or perhaps you want to get the best graphics with minimal performance impact?

This website aims to be a solution for those problems! On here, you will be able to select one of the games that have been added and compare the options for all graphics settings.

Thank you for visiting!

Contribute

You can check the repository and see how both the folder structure and JSON structure is set up, then re-create it for your own tests.

Personally, I use MSI Afterburner's benchmark feature for 20 seconds to get the FPS and Win+Shift+S for taking screenshots. When you're uploading the screenshots, only upload the non-compressed full size PNG's though, I will manually resize them to prevent too much bandwidth usage myself, so you don't have to worry about that.

Note: I will create a web based tool to make the JSON file for you, as well as the basic folder structure for games, so it should be much easier to contribute in the future!

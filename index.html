Graphics Settings Comparer

Welcome!

Have you ever wondered whether it's worth it to enable a specific graphics option in your favourite game? Or perhaps you want to get the best graphics with minimal performance impact?

This website aims to be a solution for those problems! On here, you will be able to select one of the games that have been added and compare the options for all graphics settings.

You can find a list of games by clicking the Games button over on the left. Should you be unable to find a game that you're looking for, feel free to request it.

Thank you for visiting!

It's late already and I want to go to bed soon so this is just a temporary to-do list of what I should mention here:

  • Turn off overlays (shadowplay, fraps, rtss, steam, etc.)
  • Turn off VSync and other frame limiters
  • You can either benchmark each setting via built in benchmark tool or stand still in specific spots
  • Acceptable benchmarks or standing-still-in-one-spot-scenes are only valid, if no PC parts were limited due to thermals
  • If overclock or underclock applied, mention it
  • List specs: CPU, RAM (GB per stick, frequency), GPU, and when comparing load times also the drive the game is on
  • Try taking multiple screenshots for each option, showcasing what difference it can make
